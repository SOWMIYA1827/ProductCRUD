# 🛒 Online Shopping Management System

🌐 **A React-based E-Commerce Web Application**

---

## 📖 Overview

The **Online Shopping Management System** is a React web application that provides separate dashboards for **Admin** and **Users**.

- 👨‍💼 **Admin** can manage products by adding, updating, editing, and deleting them.
- 👤 **Users** can browse products, add items to favorites and cart, and purchase products.

---

## ✨ Features

### 🔐 Authentication

- 👤 User Login
- 📝 User Registration (Signup)
- 🔑 Secure Authentication
- 🎭 Role-Based Access (Admin & User)

### 👨‍💼 Admin Panel

- ➕ Add New Products
- ✏️ Edit Product Details
- 🗑️ Delete Products
- 📦 Manage Product Inventory
- 👀 View All Products

### 👤 User Panel

- ❤️ Add to Favorites
- 🛒 Add to Cart

- 📱 Fully Responsive Design

- 🎯 Clean & Modern UI

## 🚀 Installation

Clone the repository

```bash
git clone <repository-url>
```

Go to the project folder

```bash
cd Online-Shopping-System
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npm start
```

Open your browser

```
http://localhost:3000
```
